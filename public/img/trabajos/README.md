# Imágenes para Trabajos Realizados

Este directorio almacena las imágenes de documentación de trabajos realizados en el condominio.

## Estructura

Las imágenes se organizan por trabajo y tipo:

- `[trabajo-slug]-antes.jpg` - Fotos del estado inicial
- `[trabajo-slug]-durante.jpg` - Fotos del proceso
- `[trabajo-slug]-despues.jpg` - Fotos del resultado final

## Formato Recomendado

- **Formato**: JPG o PNG
- **Tamaño**: Máximo 2MB por imagen
- **Resolución**: 1200px de ancho máximo (se redimensiona automáticamente)

## Ejemplo

Para el trabajo de tapa de cisterna (`2026-02-tapa-cisterna`):
- `tapa-cisterna-antes.jpg`
- `tapa-cisterna-durante.jpg`
- `tapa-cisterna-despues.jpg`

## Nota Importante

Por favor, agrega las imágenes reales correspondientes a cada trabajo. Las imágenes actualmente referenciadas en los MDX deben ser agregadas manualmente en este directorio.
